EAC日本語化ファイル

Exact Audio Copy (EAC) の日本語化ファイル一式です。

対応確認済みバージョン

EAC V1.5

フォルダ構成

/
├ Languages/
|  |-- Japanese.txt
├ Languages_freedb/
|  |-- Japanese.txt
├ Languages_gd3/
   |-- Japanese.txt

適用方法

Code > Download ZIP もしくはこちらからダウンロードしてください。

解凍後、EACのインストール先にLanguagesフォルダ、Languages_freedbフォルダ、Languages_gd3フォルダをコピーしてください。

EAC本体 日本語化ファイル

Languages/Japanese.txtはEAC本体の日本語化言語ファイルです。

初版は、945様、yk様、Chappu様、kazutomo(ハウツーIT)様、じゅん様が翻訳・公開されてきたファイルをリポジトリ登録しました。

freedb metadata plugin 日本語化ファイル

Languages_freedb/Japanese.txtはfreedb metadata pluginの日本語化言語ファイルです。

初版は、シャー芯(sha-shin)様が翻訳・公開されてきたファイルをリポジトリ登録しました。

GD3 metadata plugin 日本語化ファイル

Languages_gd3/Japanese.txtはGD3 metadata pluginの日本語化言語ファイルです。

初版は、シャー芯(sha-shin)様が翻訳・公開されてきたファイルをリポジトリ登録しました。
